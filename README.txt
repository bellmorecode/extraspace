This is an app called 'extraspace'

the concept is to provide two experiences, one for mobile devices 
and one for desktops.  
It should support multiple, concurrent mobile devices.
all nodes are peers. multiple nodes can be app-hosts concurrently 
in the style of a load-balanced web app.

login and connect.

interesting scenarios: phones are controllers 
		and the PC\Laptop can be the game surface.
